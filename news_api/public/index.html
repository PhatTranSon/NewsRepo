NewsRepo API

About

NewsRepo is a free-to-use RESTful API to retrieving news from authoritative sources such as the New York Times or APNews

How to use

There are several ways the application can be used with several endpoints

Gather all news

https://articlesfinder.net/news/all/?limit=10&after_id=0

This endpoint retrieves all the news scraped by our API

  • limit: (Integer) The number of news articles required
  • after_id: (Integer) Gather news after this id only (news have id from 1 to ...)

Gather news by id

https://articlesfinder.net/news/byid?id[]=1&id[]=3&id[]=...

This endpoint retrieves all the news by ids

  • id: (Array) The ids of news articles

Disclaimer

Since all the news are collected from The New York Times and APNews using scraping techniques, the API is unavailable for commericial use.
